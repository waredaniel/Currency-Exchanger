# Currency-Exchanger

#### Currency-Exchanger is a handy tool for converting between world currencies using ExchangeRate-API

#### By Daniel Ware

## Technologies Used

* HTML5
* Javascript
* jQuery
* CSS/Bootstrap
* Node.js

## Setup/Installation Requirements

* Clone repository
* Open Currency-Exchanger in VS Code
* npm install
* npm run build
* npm run start
